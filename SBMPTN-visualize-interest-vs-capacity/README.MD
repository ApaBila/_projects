Jika menggunakan kode ini untuk mengakses situs LTMPT pada tahun-tahun berikutnya, 
mohon juga mempertimbangkan traffic situs tersebut, utamakan yang perlu akses data tersebut untuk SBMPTN

This is repo contains commented code to reproduce the charts in [my blog about the importance of choosing undergraduate programs in Indonesia](https://medium.com/@salsabila.mahdi/pentingnya-dan-cara-memilih-program-studi-1c5e9482ff4d). 
There are 5 steps:
(1) Setup
(2) Get links to all university pages
(3) Scrape data from each page
(4) Process data into appropriate format
(5) Make charts

Find a [tutorial with the outputs here] (http://rpubs.com/salsabilamahdi/sbmptn)
